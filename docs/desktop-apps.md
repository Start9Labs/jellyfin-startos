**TIP:** You can access Jellyfin and play your media through a web browser, but if you prefer, you can install an app.


**Instructions**

## macOS or Windows

1. Download the .dmg file (macOS) or the .exe file (Windows) for [Jellyfin Media Player](https://github.com/jellyfin/jellyfin-media-player/releases).

1. Upon opening the client, you will be prompted to add a server. Click "Add Server".

1. Next, you will be prompted for a server address. Paste your preferred interface URL from Jellyfin service interfaces (`Services -> Jellyfin -> Interfaces`) and click "Connect".

1. Next, enter your username and password and click "Sign In".
